# Link-in-Bio Generator

**Ledger & Spark** — Free tools. No catch. Built because you asked.

## What It Does
Pick a vibe, add your links, download a beautiful link-in-bio page you own forever. Free alternative to Linktree ($24/mo).

## How to Use
1. Pick your vibe (Clean, Neon, Pastel, Bold)
2. Enter your name, bio, and up to 5 links
3. Pick your accent color
4. Generate → preview → copy HTML or download .html

## Tech
Single HTML file. No backend. No dependencies. All CSS/JS inline.
